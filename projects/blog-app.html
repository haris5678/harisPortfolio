Blog App

A Full-stack application developed using React, Node and PostgreSQL

Project Overview

The Blog App project was dynamic full-stack application developed using React for the frontend and Node.js for the backend. It provides a feature-rich platform for users to create and manage their blogs.
The project implementation include comprehensive authentication functionalities such as user registration, email confirmation, login, and profile modification.
The system implements user roles, including Admin, Moderators, and Users, with specific privileges and responsibilities assigned to each role. The dashboard page offers a concise overview of recent posts, comments, and likes. Users can create, view, and style their blog posts with support for inline attachments, which are uploaded to a cloud storage service like S3. Posts can be liked, commented on, or reported by users. The comments section allows attachments, replies, and likes.
Additionally, the app incorporates a suggestions feature, enabling users to check, modify, reject, reply to suggestions, and view their own submitted suggestions on others' posts. Moderators have the authority to approve user-submitted posts and manage reported posts, while the admin section utilizes RailsAdmin to facilitate comprehensive management of users, posts, comments, likes, reports, and suggestions.
The Blog App project showcases expertise in React, Node.js, authentication, user roles, dynamic content creation, and robust administrative capabilities, offering a powerful platform for bloggers and content management.
