E-commerce

A Full-stack application developed using React, Node and PostgreSQL

Project Overview

The E-commerce project is a full-stack application built with React for the frontend and Node.js for the backend. Its primary objective is to create an online shopping platform with various essential modules.
Users can sign up, sign in, and personalize their profiles with display pictures. The Products module enables users to create, edit, and delete products, each with multiple images and a unique serial number. The Comments feature allows users to leave feedback on products, with real-time updating and a list of comments displayed on the product show page. The Cart module allows users to add, edit quantities, and remove products, and checkout requires user authentication.
Stripe is integrated as the payment method, ensuring secure transactions. The system maintains user actions related to orders and payments. The Search functionality includes a search field in the header, enabling users to find products, with search results displayed and the ability to click and be redirected to the product show page.
The E-commerce project effectively demonstrates proficiency in React, Node.js, database management, and delivers a comprehensive E-commerce solution with essential features for seamless online shopping experiences.
