Instagram Clone

A Full-stack application developed using React and Ruby on Rails.

Project Overview

The Instagram Clone project is a full-stack application developed using React and Ruby on Rails. It replicates the core features of Instagram, allowing users to sign up, create posts, share stories, leave comments, and follow other users.
Users can sign in, upload a display picture, and edit/delete their own posts. Each post can contain up to 10 images. Stories remain visible for 24 hours and can be deleted by the user. Users can comment on their own and other posts, edit/delete their own comments, and like posts. The follower system enables users to follow each other, with the option to set accounts as private or public.
The application includes a search feature for finding other users, with search results displayed below the search field. The project showcases proficiency in React, Ruby on Rails, and database management, successfully replicating the core functionalities of Instagram.
